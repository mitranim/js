## Overview

[dom_shim.mjs](../dom_shim.mjs) provides a lightweight and performant shim for DOM node and element classes. Allows server-side rendering with [`prax`](prax_readme.md).

## TOC

* [#Usage](#usage)
* [#API](#api)
  * [#Undocumented](#undocumented)

## Usage

```js
import * as ds from 'https://cdn.jsdelivr.net/npm/@mitranim/js@0.1.64/dom_shim.mjs'
```
