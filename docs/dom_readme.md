## Overview

[dom.mjs](../dom.mjs) provides tiny utils and shortcuts for working with the DOM.

## TOC

## Usage

```js
import * as d from 'https://cdn.jsdelivr.net/npm/@mitranim/js@0.1.39/dom.mjs'
```
