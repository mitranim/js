## Overview

[time.mjs](../time.mjs) provides tools for datetime manipulation. Stuff missing from JS built-ins.

## TOC

* [#Usage](#usage)
* [#API](#api)
  * [#Undocumented](#undocumented)

## Usage

```js
import * as ti from 'https://cdn.jsdelivr.net/npm/@mitranim/js@0.1.53/time.mjs'
```
